# Python - Manipulação de Arquivos (Parte 1)

## 📖 Objetivo

Aplicar os conceitos de manipulação de arquivos em Python, realizando operações básicas de leitura, escrita e armazenamento de dados.

## 📂 Conteúdo

O repositório contém exercícios sobre:

- Leitura de arquivos;
- Escrita em arquivos;
- Criação de arquivos;
- Manipulação de dados em arquivos texto.

## 💻 Tecnologias

- Python 3

## ▶️ Como executar

1. Instale o Python 3 em seu computador.
2. Clone este repositório ou faça o download dos arquivos.
3. Execute o arquivo desejado pelo terminal:

```bash
python nome_do_arquivo.py
```
